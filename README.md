# sql-data-warehouse
Data Warehouse and Analytics Project respository.

Project involves:
1. Data Architecture: Designing a Data Warehouse using Medalion Architecture Bronze, Silver and Gold layers.
2. ETL Pipelines: Extracting, transforming and loading data from source systems into the warehouse.
3. Data Modeling: Developing fact and dimension tables.
4. Data Analytics: Set of SQL scripts for data exploration, analytics, and reporting.

Data Architecture
1. Bronze Layer: Stores raw data as-is from the source systems. Data is ingested from CSV files into SQL Server Database
2. Silver Layer: This layer includes cleansing, standarization and normalization processes to perpare data for analysis
3. Gold Layer: Stores buisness-ready data
